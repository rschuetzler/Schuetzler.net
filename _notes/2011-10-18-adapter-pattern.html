The Problem: An object has the right behavior, but the wrong interface

The Solution: The Adapter Pattern

This pattern is very similar to the Facade, but specifically adapted for one class. A wrapper is written for the class that has the wrong interface, with the wrapper providing the translation from the correct interface to the actual class being adapted. Fairly simple.
